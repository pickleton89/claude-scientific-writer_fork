# Output Templates

> Ready-to-use templates for slide planning and timing documentation.

---

## Slide Plan Template

```markdown
# Presentation: [TITLE]
Duration: [X] minutes | Slides: [N] | Type: [conference/seminar/defense/grant/journal]

## Formatting Goal
Background: [color]
Text: [color]
Accent: [color]
Style: [minimal/corporate/academic]

## Slide 1: Title
- Title: "[Full title]"
- Subtitle: "[Conference/Event]"
- Speaker: [Name, Affiliation]
- Visual: [Background description]

## Slide 2: Hook
- Title: "[Compelling question or statement]"
- Key point: [What grabs attention]
- Citations: (Author1 et al., Year; Author2 et al., Year)
- Visual: [Image/diagram description]

## Slide 3: Background
- Title: "[Context title]"
- Points: [2-3 bullet points]
- Citations: [Key papers]
- Visual: [Diagram/image]

## Slide 4: Methods
- Title: "[Approach title]"
- Key points: [2-3 methodological highlights]
- Visual: [Workflow diagram or key figure]

## Slide 5-N: Results
- Title: "[Finding title]"
- Key finding: [One main takeaway]
- Visual: [Data figure - dominant element]
- Annotations: [2-3 bullet points explaining figure]

## Slide N+1: Discussion
- Title: "[Implications title]"
- Points: [2-3 key implications]
- Citations: [Comparison papers]
- Visual: [Summary diagram or comparison]

## Slide N+2: Conclusions
- Title: "Key Takeaways" or "Summary"
- Points: [3-4 main conclusions, matching intro promises]
- Visual: [Memorable closing image or diagram]

## Slide N+3: Acknowledgments
- Funding sources
- Collaborators
- Contact information

## Backup Slides
- [Topic 1]: For Q&A about [specific topic]
- [Topic 2]: Additional data if asked
- [Topic 3]: Methodology details if questioned
```

## Timing Checkpoint Template

```markdown
# Timing Checkpoints: [TITLE]
Total duration: [X] minutes

| Checkpoint | Target Time | Actual Run 1 | Actual Run 2 | Actual Run 3 |
|------------|-------------|--------------|--------------|--------------|
| End of intro | [X:XX] | | | |
| Start of results | [X:XX] | | | |
| Halfway through results | [X:XX] | | | |
| Start of conclusions | [X:XX] | | | |
| End | [X:XX] | | | |

Notes:
- Run 1: [observations]
- Run 2: [observations]
- Run 3: [observations]
```

## Issue Tracking Template

```markdown
# Visual Review: [TITLE]
Date: [YYYY-MM-DD]

| Slide # | Issue | Priority | Status |
|---------|-------|----------|--------|
| 3 | Text overflow in bullet 4 | High | Fixed |
| 7 | Figure overlaps caption | High | Pending |
| 12 | Font too small (18pt) | Medium | Fixed |

Resolution Notes:
- Slide 3: Reduced text, regenerated with shorter bullets
- Slide 7: [action taken]
```
